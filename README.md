# Employee Management System (EMS) Backend

A Spring Boot REST API for managing employee records.  
Features MySQL integration, Hibernate ORM, and Postman-tested endpoints.

## Tech Stack
- Java
- Spring Boot
- Hibernate
- MySQL
- Postman

## Features
- CRUD operations for employees
- RESTful API endpoints
- Console and SQL log verification
- MySQL Workbench integration
