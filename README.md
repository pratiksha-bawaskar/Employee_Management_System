# Employee Management System (EMS) Backend

A Spring Boot REST API for managing employee records with full CRUD operations.  
This project uses MySQL for database storage, Hibernate ORM for data persistence, and Postman for API testing.

## Tech Stack
- Java
- Spring Boot
- Hibernate (JPA)
- MySQL
- Postman

## Features
- Create, update, delete, and view employees
- RESTful API endpoints
- Layered architecture (Controller, Service, Repository)
- Data verification using console logs and SQL queries
- MySQL Workbench integration for database management

## How to Run the Project
1. Clone this repository
2. Open the project in IntelliJ or Eclipse
3. Configure MySQL database in `application.properties`
4. Run the application as a Spring Boot App
5. Test APIs using Postman
